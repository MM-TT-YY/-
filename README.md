## 游戏玩法

- 通过发出**超出阈值的声音**（如"哈气"）控制耄耋跳跃
- 成功躲避障碍可获得**火腿肠**奖励


## 安装与运行

1. 确保已安装Python 3.x，无需python环境的exe和apk版本正在开发中
2. 安装依赖库：
   ```bash
   pip install pygame
   pip install numpy
   pip install sounddevice
